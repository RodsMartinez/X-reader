# X-reader

X-reader is a document and text-reading application designed to provide users with an intuitive and efficient way to view, manage, and interact with digital text. This project was developed as a final project and demonstrates software engineering principles, user interface design, and file management.

## Features

- 📄 Open and read supported document files
- 🔍 Search within documents
- 📂 Simple file management
- 💡 Clean and user-friendly interface
- ⚡ Fast and responsive performance

## Installation

Clone the repository:

```bash
git clone https://github.com/RodsMartinez/X-reader.git
```

Go to the project directory:

```bash
cd X-reader
```

Install the required dependencies (if any):

```bash
# Example
npm install

# or
pip install -r requirements.txt
```

Run the project:

```bash
# Example
npm start

# or
python main.py
```
